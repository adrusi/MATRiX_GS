MATRiX_GS : SCSS-Based Grid System

How to Use

When you download MATRiX, the first step to using it is to make sure the file is named "_grid.scss". Import it into your SCSS stylesheets using @import "grid";. Importing it in this way makes sure that you get all of the mixins and variables from the file. The next step is to set all of the defaults. The first variables set the dimensions of the grid.

$container-width: 960px;
$column-count: 12;
$column-margin: 20px;

The purpose of each is fairly self explanatory, and you can set them to whatever you want. The next set of variables set the default options for grid elements. Leave the values as they are unless you are creating a really weird layout and know exactly what you'r doing.

$_span: 1;
$_border-width: 0;
$_border-left: 0;
$_border-right: 0;
$_border-style: outset;
$_left-padding: 0;
$_right-padding: 0;
$_prepend: 0;
$_append: 0;
$_shift: 0;

Finally you need to initialize MATRiX by including the init mixin outside of any rule set. @include init;. By default, the init mixin adds the HTML5doctor Reset, you can choose not to include this by passing any argument to it other than "reset". Now you're ready to use MATRiX. Set the container element by adding the following line to the element you want to be the container: @include container;. If you want to see the background grid behind the container, pass a color the the mixin like this: @include container(green);. This only works in webkit and gecko.

The most basic grid elements just span a set number of columns. This is what the $span variable is for. If you set $span to a number, the element will span that number of columns. For example, $span: 4; would make the element span 4 columns. You can set other attributes of grid elements in a similar manner using the guide below to see what capabilities there are and how to implement them.

After you set all the attributes of a grid element, you need to call the grid mixin using @include grid;. If the element is the first child of another grid element, pass "first" to the mixin (no quotes): @include grid(first);.

Span

$span: 1; $span: 11; $span: 2; $span: 10; $span: 3; $span: 9; $span: 4; $span: 8; $span: 5; $span: 7; $span: 6; $span: 6;

Append

$span: 6; $append: 1; $span: 5; $span: 6; $append: 2; $span: 4; $span: 6; $append: 3; $span: 3;

Prepend

$span: 5; $prepend: 1; $span: 6; $span: 4; $prepend: 2; $span: 6; $span: 3; $prepend: 3; $span: 6;

Shift (Relative Positioning)

$span: 3; $shift: 9; $span: 9; $shift: -3; $span: 4; $shift: 8; $span: 8; $shift: -4; $span: 5; $shift: 7; $span: 7; $shift: -5;

Outset Border

$span: 6; $border-width: 10px; ... border: 10px solid #888888; $span: 6; ... margin-bottom: 20px; $span: 6; $border-width: 9px; ... border: 9px solid #888888; $span: 6; $border-width: 9px; ... border: 9px solid #888888;

Inset Border (elements with inset borders cannot contain other grid elements)

$span: 6; $border-width: 10px; $border-style: inset; ... $span: 6; ... margin-bottom: 20px; $span: 6; $border-width: 9px; $border-style: inset; ... $span: 6; $border-width: 9px; $border-style: inset; ...

Padding

$span: 5; $right-padding: 1; $span: 6; $span: 3; $left-padding: 3; $span: 6;

Padding and Borders

$span: 5; $right-padding: 2; $border-width: 5px; ... $span: 5; ... margin-bottom: 10px; $span: 7; $right-padding: 2; $border-width: 5px; $border-style: inset; ... $span: 3; ... margin-bottom: 10px;
